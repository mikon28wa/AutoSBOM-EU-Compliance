# AutoSBOM-EU-complete
 SBOM-Generator zur Erstellung von Software Bill of Materials (SBOM) mit starkem Fokus auf EU-Compliance.   Unterstützt SPDX 3.0.1, CycloneDX 1.6, Syft/Grype Integration, Nested SBOMs und CRA/BSI-konforme Ausgaben.   Entwickelt durch eine detaillierte Systemanweisung für AI-Coding-Tools. Perfekt für regulierte Umgebungen und sichere Lieferketten.
